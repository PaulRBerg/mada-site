# MADA - The Squirrel Catalog

An interactive catalog of squirrel species from around the world. Explore tree squirrels, ground squirrels, flying squirrels, chipmunks, and marmots with detailed information about their characteristics, habitats, and behaviors.

## Features

- **15+ Species** across 5 categories (tree squirrels, ground squirrels, flying squirrels, chipmunks, marmots)
- **Filter by category** or geographic region (North America, Europe, Asia, Africa, South America, Oceania)
- **Species cards** with physical measurements, diet, habitat, activity patterns, and conservation status
- **Wikipedia links** for each species
- **Responsive design** with dark mode support

## Getting Started

### Prerequisites

- [Bun](https://bun.sh)
- [Ni](https://github.com/antfu-collective/ni)
- [Just](https://just.systems)

### Installation

```bash
bun install
```

### Development

```bash
just dev
```

The dev server starts on a random available port. Check the terminal output for the URL.

### Build

```bash
just build
just start
```

## Tech Stack

- [Next.js 16](https://nextjs.org) with App Router and React 19
- [TypeScript](https://typescriptlang.org)
- [Tailwind CSS 4](https://tailwindcss.com)
- [Lucide React](https://lucide.dev) for icons
